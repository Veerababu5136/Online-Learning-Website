C++ Programming

What is C++?

C++ is a general-purpose programming language that was developed in the 1970s. It is a popular language for system programming and is used to develop operating systems, compilers, and other low-level software.

Data Types

C++ has several built-in data types, including:

  • int
  • float
  • double
  • char

Variables

In C++, variables are used to store data values. To declare a variable, you need to specify the data type and a name for the variable.

      int age;
      float temperature;
      char letter;

Control Structures

C++ provides several control structures, including:

  • if/else
  • for
  • while
  • switch

C++ Variables

Variable Definition

A variable definition in C++ programming language declares a variable and specifies its data type, and optionally initializes it with a value. For example:

      int number;
      float price;
      char letter;

Variable Declaration

A variable declaration in C++ programming language declares a variable and specifies its data type, but does not initialize it with a value. For example:

      int number;
      float price;
      char letter;

Variable Initialization

A variable initialization in C++ programming language declares a variable, specifies its data type, and initializes it with a value. For example:

      int number = 10;
      float price = 2.99;
      char letter = 'a';

Variable Scope

A variable's scope in C++ programming language defines the part of the program where the variable can be accessed. There are three types of variable scope:

  • Global scope
  • Local scope
  • Function parameter scope

C++ Datatypes

Basic Datatypes

In C++ programming language, there are four basic datatypes:

  • int
  • float
  • double
  • char

Type Modifiers

In C++ programming language, there are four type modifiers:

  • signed
  • unsigned
  • short
  • long

Derived Datatypes

In C++ programming language, there are two derived datatypes:

  • Array
  • Pointer

C++ Operators

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on operands.

    + (addition)
    - (subtraction)
    * (multiplication)
    / (division)
    % (modulus)

Assignment Operators

Assignment operators are used to assign values to variables.

    =  (simple assignment)
    += (add and assign)
    -= (subtract and assign)
    *= (multiply and assign)
    /= (divide and assign)
    %= (modulus and assign)

Relational Operators

Relational operators are used to compare two operands.

    == (equal to)
    != (not equal to)
    < (less than)
    <= (less than or equal to)
    > (greater than)
    >= (greater than or equal to)

Logical Operators

Logical operators are used to perform logical operations on operands.

    && (logical AND)
    || (logical OR)
    ! (logical NOT)

Bitwise Operators

Bitwise operators are used to perform bitwise operations on operands.

    & (bitwise AND)
    | (bitwise OR)
    ^ (bitwise XOR)
    ~ (bitwise NOT)
    << (left shift)
    >> (right shift)

Ternary Operator

The ternary operator is a shorthand way of writing an if-else statement.

    variable = (condition) ? value1 : value2;

Sizeof Operator

sizeOf operator is used to find the size or memory allocated to a varible,expression

C++ Decision Making Statements

If Statement

The if statement is used to execute a block of code if a certain condition is true. The syntax of the if statement is as follows:

      if (condition) {
        // code to be executed if condition is true
      }

If-Else Statement

The if-else statement is used to execute one block of code if a certain condition is true and another block of code if the condition is false. The syntax of the if-else statement is as follows:

      if (condition) {
        // code to be executed if condition is true
      } else {
        // code to be executed if condition is false
      }

Nested If Statement

The nested if statement is used when an if statement is needed inside another if statement. The syntax of the nested if statement is as follows:

      if (condition1) {
        if (condition2) {
          // code to be executed if both conditions are true
        }
      }

Switch Statement

The switch statement is used to execute different blocks of code depending on the value of a variable. The syntax of the switch statement is as follows:

      switch (expression) {
        case value1:
          // code to be executed if expression equals value1
          break;
        case value2:
          // code to be executed if expression equals value2
          break;
        ...
        default:
          // code to be executed if expression does not equal any of the values
          break;
      }

C++ Looping Statements

While Loop

The while loop is used to execute a block of code as long as a certain condition is true. The syntax of the while loop is as follows:

      while (condition) {
        // code to be executed while condition is true
      }

Do-While Loop

The do-while loop is similar to the while loop, but the condition is checked at the end of the loop instead of at the beginning. This ensures that the block of code is executed at least once. The syntax of the do-while loop is as follows:

      do {
        // code to be executed at least once
      } while (condition);

For Loop

The for loop is used to execute a block of code for a specified number of times. The syntax of the for loop is as follows:

      for (initialization; condition; increment/decrement) {
        // code to be executed for each iteration
      }

Nested Loop

The nested loop is used when a loop is needed inside another loop. The syntax of the nested loop is as follows:

      for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 3; j++) {
          // code to be executed for each combination of i and j
        }
      }
