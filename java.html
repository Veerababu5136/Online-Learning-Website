Java Learning

Introduction

Java is a high-level programming language developed by Sun Microsystems. It is a popular language for developing desktop, web, and mobile applications.

Variables

In Java, variables are used to store data values. A variable has a name, a data type, and a value.

Data Types

Java has two categories of data types: primitive and non-primitive. Primitive data types are the basic data types, while non-primitive data types are the reference types.

Loops

Loops are used to repeat a block of code until a condition is met. Java has three types of loops: for, while, and do-while.

Arrays

An array is a collection of elements of the same data type. In Java, arrays can be of any data type, including primitive and non-primitive data types.

Java Variables

int

An int is a data type used to store integers, which are whole numbers without decimal points. The range of values that can be stored in an int is -2147483648 to 2147483647.

double

A double is a data type used to store floating-point numbers, which are numbers with decimal points. The range of values that can be stored in a double is approximately ±4.9 x 10^-324 to ±1.8 x 10^308.

boolean

A boolean is a data type used to store either true or false. Boolean values are often used in conditional statements to control the flow of a program.

Java Data Types

Primitive Data Types

Java has eight primitive data types:

  • byte
  • short
  • int
  • long
  • float
  • double
  • char
  • boolean

Each primitive data type has a specific range of values it can hold and a default value.

Non-Primitive Data Types

Java has four non-primitive data types:

  • String
  • Arrays
  • Classes
  • Interfaces

Non-primitive data types are also called reference types because they refer to objects.

Java Operators

Arithmetic Operators

The arithmetic operators are used to perform basic mathematical operations such as addition, subtraction, multiplication, and division.

  • +
  • -
  • *
  • /
  • %

Assignment Operators

The assignment operators are used to assign values to variables.

  • =
  • +=
  • -=
  • *=
  • /=
  • %=

Comparison Operators

The comparison operators are used to compare two values and return a boolean value of either true or false.

  • ==
  • !=
  • <
  • >
  • <=
  • >=

Logical Operators

The logical operators are used to combine multiple boolean expressions and return a boolean value of either true or false.

  • &&
  • ||
  • !

Java Operators Examples

Arithmetic Operators Examples

Here are some examples of using arithmetic operators:

  • 5 + 3 = 8
  • 7 - 2 = 5
  • 4 * 2 = 8
  • 10 / 2 = 5
  • 9 % 2 = 1

Assignment Operators Examples

Here are some examples of using assignment operators:

  • int x = 5;
  • x += 3; // x is now 8
  • x -= 2; // x is now 6
  • x *= 2; // x is now 12
  • x /= 3; // x is now 4
  • x %= 2; // x is now 0

Comparison Operators Examples

Here are some examples of using comparison operators:

  • 5 == 5 // true
  • 5 != 5 // false
  • 7 < 10 // true
  • 7 > 10 // false
  • 5 <= 5 // true
  • 5 >= 5 // true

Logical Operators Examples

Here are some examples of using logical operators:

  • (5 > 3) && (7 < 10) // true
  • (5 < 3) || (7 > 10) // false
  • !(5 < 3) // true

Java Control Statements

If-else Statements Example

Here's an example of using if-else statements:

int age = 18;
if (age >= 18)
{
System.out.println("You are an adult");
} 
else 
{
System.out.println("You are not yet an adult");
}

Switch Statements Example

Here's an example of using switch statements:

String color = "red";
switch (color)
 {
case "red":
System.out.println("The color is red");
break;
            
case "blue":
System.out.println("The color is blue");
break;
            
case "green":
System.out.println("The color is green");
break;
            
default:
System.out.println("The color is unknown");
break;
}

For Loop Example

Here's an example of using a for loop:

 for (int i = 0; i < 5; i++) 
 {
System.out.println("The value of i is: " + i);
}

While Loop Example

Here's an example of using a while loop:

int i = 0;
while (i < 5)
{
System.out.println("The value of i is: " + i);
i++;
}

Do-while Loop Example

Here's an example of using a do-while loop:

int i = 0;
do 
{
System.out.println("The value of i is: " + i);
i++;
} while (i < 5);
