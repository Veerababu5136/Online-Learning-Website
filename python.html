Python Learning

Introduction to Python

Python is a popular programming language that is easy to learn and widely used in various fields including web development, data science, and artificial intelligence. In this lesson, you will learn the basics of Python programming.

Data Types and Variables

In this lesson, you will learn about the basic data types in Python such as integers, floats, strings, and booleans, and how to declare and manipulate variables.

Control Structures

Control structures are used to control the flow of execution in a program. In this lesson, you will learn about conditional statements, loops, and functions in Python.

Python Variables

A variable is a named location in memory used to store a value. In Python, variables are created when you assign a value to them.

For example:

x = 10
y = "hello"

The first line creates a variable named x and assigns it the value 10. The second line creates a variable named y and assigns it the value "hello".

Variables can also be reassigned with a new value:

x = 20
y = "world"

Now, the value of x is 20 and the value of y is "world".

Python variables can store different types of data, such as:

  • Integers
  • Floats
  • Strings
  • Booleans
  • Lists
  • Tuples
  • Dictionaries

Python Datatypes

Introduction

Python is a dynamically typed language, which means that the type of a variable is determined at runtime. Python has several built-in datatypes that can be used to represent various types of data, including:

  • Strings (str)
  • Integers (int)
  • Floating-point numbers (float)
  • Lists (list)
  • Tuples (tuple)
  • Dictionaries (dict)
  • Sets (set)
  • Booleans (bool)
  • NoneType (None)

Examples

Here are some examples of Python code that use these datatypes:

Code Output
my_str = "Hello, world!"
print(my_str)		 Hello, world!
my_int = 42
print(my_int)		 42
my_float = 3.14
print(my_float)		 3.14
my_list = [1, 2, 3]
print(my_list)		 [1, 2, 3]
my_dict = {"a": 1, "b": 2, "c": 3}
print(my_dict)		 {"a": 1, "b": 2, "c": 3}

Python Operators

Introduction

Operators are special symbols in Python that perform arithmetic or logical operations on operands. Python supports several types of operators, including:

  • Arithmetic operators (+, -, *, /, %, //, **)
  • Comparison operators (==, !=, <, >, <=, >=)
  • Logical operators (and, or, not)
  • Bitwise operators (&, |, ^, ~, <<, >>)
  • Assignment operators (=, +=, -=, *=, /=, %=, //=, **=, &=, |=, ^=, ~=, <<=, >>=)
  • Identity operators (is, is not)
  • Membership operators (in, not in)

Examples

Here are some examples of Python code that use these operators:

Code Output
print(3 + 4) 7
print(5 * 6) 30
print(10 % 3) 1
print(5 == 5) True
print(10 < 5) False
print(True and False) False
my_list = [1, 2, 3]
print(2 in my_list)		 True
